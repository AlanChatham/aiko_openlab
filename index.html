aiko_openlab by geekscape

Educational application with graphical LCD for general purpose input / output, waveform generation and display.

OpenLab software design overview (2011-05-04)

Dependencies

Aiko framework

GLCD library (modified)

Install

    • 1) Download Arduino IDE
      2) mkdir -p ~/arduino/sketchbook/libraries
      3) cd ~/arduino/sketchbook/libraries
      4) git clone https://github.com/geekscape/Aiko.git
      5) cd ~/arduino/sketchbook
      6) git clone https://github.com/geekscape/aiko_openlab.git
      7) cd ~/arduino/sketchbook/libraries
      8) tar -zxpf ../aiko_openlab/libraries/glcd_shiftRegister.tgz
